QDM 홈페이지 전체 업로드용 패키지

포함 내용:
- index.html
- 포트폴리오 상세페이지 4개
- portfolio.html
- assets/style.css
- assets/site.js
- assets 이미지 전체
- data/blog-posts.json
- data/portfolios.json

업로드 방법:
1. GitHub qdm-homepage 저장소에 있는 기존 파일을 기준으로 전체 교체합니다.
2. 이 압축파일 안의 모든 파일/폴더를 그대로 업로드합니다.
3. Commit changes를 누르면 Vercel이 자동 배포합니다.

주의:
- assets 폴더가 빠지면 포트폴리오 상세페이지 CSS와 이미지가 깨집니다.
- data/blog-posts.json은 최신 기술자료 관리용입니다.
- data/portfolios.json은 포트폴리오 관리용입니다.
